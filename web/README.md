# Artify Web - Frontend

Frontend Next.js application for Artify Paris cultural events platform.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm

### Installation

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Environment Variables

Create a `.env.local` file:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Features

- Event listing with filters
- Event detail pages
- Free event badges
- Price display
- Responsive design
- Server-side rendering
